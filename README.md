# Thesis Code Repository

Code and experiments for my thesis: Understanding Soil Moisture Predictability in the Ebro River Basin Using Machine Learning and Explainable AI

## Overview
This repository contains the code used for data processing, experiments, and evaluation in my thesis.

## Project Structure
- `notebooks/` — Jupyter notebooks for data exploration and experiments
- `data/` — dataset
- `figures/` — plots used in the thesis
